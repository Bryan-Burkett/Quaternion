2.1 Quaternion

struct

(struct quaternion (h i j k)
  #:extra-constructor-name make-quaternion)
  h : real?
  i : real?
  j : real?
  k : real?
A structure to represent a quaternion. h represents the real part, i, j and k represent the imaginary parts. If not provided with 4 real numbers, return a type-wrong-format error. Automatically defines (quaternion? x) to test if x is a quaternion. (quaternion-h x), (quaternion-i x), (quaternion-j x) and (quaternion-k x) can be used to get the components of a quaternion x.

procedure

(eq? x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Defined in the module as (quaternion-equal number1 number2). Returns #t if the two arguments are equevilent mathematically, otherwise returns #f. If none of the arguments are quaternions, uses racket’s default eq? function, otherwise it converts each argument into a quaternion, then compares each corresponding element of two quaternions left to right. The and of those comparisions is returned.
>(eq? (quaternion 2 5 34 -1) (quaternion 2 5 34 -1))
#t
>(eq? (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
#f

procedure

(quaternion->string x) → string?

  x : quaternion?/number?
Returns x as a string representation of a quaternion. Given in the form of "(quaternion h i j k)".
>(quaternion->string (quaternion 2 5 34 -1))
"(quaternion 2 5 34 -1)"

procedure

(make-quaternion x) → quaternion?

  x : quaternion?/number?
Only available within the module. Turns any number into a quaternion. If x is a quaternion, returns x. If x is a number, returns (quaternion (real-part x) (imag-part x) 0 0). If neither, returns quaternion 0 0 0 0.
>(make-quaternion 2+3i)
(quaternion 2 3 0 0)

procedure

(unmake-quaternion x) → number?/quaternion?

  x : quaternion?
Only available within the module. If (quaternion-j x) and (quaternion-k x) are zero, returns a complex number that is mathematically equivalent to x, otherwise returns x.
>(unmake-quaternion (quaternion 2 3 0 0))
2+3i

procedure

(scalar-part x) → number?

  x : quaternion?/number?
Returns the real part of x as a real number. For quaternions it is equivalent to (quaternion-h x).
>(scalar-part (quaternion 2 5 34 -1))
2

procedure

(vector-part x) → quaternion?

  x : quaternion?/number?
Returns the imaginary part(s) of x as a quaternion with real part 0. It does this by converting the input to a quaternion, then subtracting the real part.
>(vector-part (quaternion 2 5 34 -1))
(quaternion 0 5 34 -1)

procedure

(norm x) → number?

  x : quaternion?
Defined within the module as (quaternion-norm number). Returns the norm of x. Also known as the magnitude, this represents the lenghth of the quaternion when represented as a vector. Found by taking the square root of the sum of the squares of the components of x.
>(norm (quaternion 2 5 34 -1))
34.4383507154451252

procedure

(conjugate x) → quaternion?/number?

  x : quaternion?/number?
Defined within the module as (quaternion-conjugate number). Returns the conjugate of x. This is a new quaternion with the same real part as x and the opposite vector parts. The product of x and it’s reciprocal is the square of the norm. This a new quaternion with the same real part as x This is equivalent to (- (real-part x) (vector-part x)), however this function takes the opposite of each imaginary component individually. If none of the arguments are quaternions, uses racket’s default conjugate procedure.
>(conjugate (quaternion 2 5 34 -1))
(quaternion  2 -5 -34 1)

procedure

(reciprocal x) → quaternion?/number?

  x : quaternion?/number?
Defined within the module as (quaternion-reciprocal number). Returns the reciprocal of x. A reciprocal is defined such that the reciprocal of x times x is 1. This procedure returns a new quaternion whose elements are the corresponding elements of x is divided by the norm of x squared.
>(reciprocal (quaternion 2 5 34 -1))
(quaternion 0.0016863406408094439 -0.004215851602023609 -0.028667790893760547 0.0008431703204047219)

procedure

(unit w) → number?

  w : quaternion?
Defined within the module as (quaternion-unit number). Returns the unit quaternion of w. This is the quaternion divided by its norm, it represents the direction of the quaternion with a parallel quaternion of norm 1.
>(unit (quaternion 2 5 34 -1))
(quaternion 0.0016863406408094439 -0.004215851602023609 -0.028667790893760547 0.0008431703204047219)

2.2 Basic Operators

procedure

(+ x ...) → quaternion?/number?

  x : quaternion?/number?
Defined within the module as (quaternion-add . quaternions). Returns the sum of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default + procedure. Otherwise, converts all arguments to quaternions and returns a new quaternion whose components are the the sums of the corresponding components of the arguments.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)
> (+ (quaternion 1 2 3 4) 4 (quaternion 4 3 5 -3) 3-2i)
(quaternion 12 3 8 1)

procedure

(- x) → quaternion?/number?

  x : quaternion?/number?
(- x w ...) → quaternion?/number?
  x : quaternion?/number?
  w : quaternion?/number?
Defined within the module as (quaternion-subtract . quaternions). Returns the difference of w from x. If no ws are supplied, returns (- 0 x) For multiple ws takes the difference of each w from x. If none of the parameters are quaternions, uses racket’s default - procedure, otherwise makes each argument a quaternion, then returns a new quaternion whose elements are the correspoonding elements of x minus the corresponding element of each w.
> (- (quaternion 1 2 3 4))
(quaternion -1 -2 -3 -4)
> (- (quaternion 3 2 4 5) (quaternion 5 3 7 5))
(quaternion -2 -1 -3 0)
> (- (quaternion 10 6 22 -5) 4 (quaternion 2 3 4 1) 3+2i)
(quaternion 1 1 18 -6)

procedure

(quaternion-multiply2 x w) → quaternion?

  x : quaternion?
  w : quaternion?
Only defined within the module, used by the * and / procedures. Returns a quaternion product of two quaternions using the following formulae:
hnew = hx*hw - ix*iw - jx*jw - kx*kw
inew = hx*iw + ix*hw + jx*kw - kx*jw
jnew = hx*jw - ix*kw + jx*hw + kx*iw
knew = hx*kw + ix*jw - jx*iw + kx*hw
}

procedure

(* x ...) → quaternion?/number?

  x : quaternion?/number?
Defined within the module as (quaternion-multiply . quaternions). Returns the product of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default * procedure. Otherwise, converts all arguments to quaternions and uses quaternion-multiply2 to multiply all of the arguments pairwise from left to right.
> (* (quaternion 1 2 3 4) (quaternion 4 3 2 1))
(quaternion -12 6 24 12)
> (* (quaternion 1 2 3 4) 2 (quaternion 4 3 2 1) 4-8i)
(quaternion 0 240 0 480)

procedure

(/ x) → quaternion?/number?

  x : quaternion?/number?
(/ x w ...) → quaternion?/number?
  x : quaternion?/number?
  w : quaternion?/number?
Defined within the module as (quaternion-divide . quaternions). Returns x divided by w. If no ws are supplied returns (reciprocal x), which is equivalent to (/ 1 x). If none of the parameters are quaternions, uses racket’s default / procedure. Otherwise.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

2.3 Advanced Operators

procedure

(exp x) → quaternion?/number?

  x : quaternion?/number?
Returns the Euler’s number to the power of x. If x is not a quaternion, uses racket’s default exp procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(expt x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Returns the x to the power of y. If if neither argument is a quaternion, uses racket’s default expt procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(sin x) → quaternion?/number?

  x : quaternion?/number?
Returns the sine of x. If x is not a quaternion, uses racket’s default sin procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(cos x) → quaternion?/number?

  x : quaternion?/number?
Returns the cosine of x. If x is not a quaternion, uses racket’s default cos procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)
