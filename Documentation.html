2.1 Quaternion

struct

(struct quaternion (h i j k)
  #:extra-constructor-name make-quaternion)
  h : real?
  i : real?
  j : real?
  k : real?
A structure to represent a quaternion. h represents the real part, i, j and k represent the imaginary parts. If not provided with 4 real numbers, return a type-wrong-format error. Automatically defines (quaternion? x) to test if x is a quaternion. (quaternion-h x), (quaternion-i x), (quaternion-j x) and (quaternion-k x) can be used to get the components of a quaternion x.

procedure

(eq? x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Defined in the module as (quaternion-equal number1 number2). Returns #t if the two arguments are equevilent mathematically, otherwise returns #f. If none of the arguments are quaternions, uses racket’s default eq? function, otherwise it converts each argument into a quaternion, then compares each corresponding element of two quaternions left to right. The and of those comparisions is returned.
>(eq? (quaternion 2 5 34 -1) (quaternion 2 5 34 -1))
#t
>(eq? (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
#f

procedure

(quaternion->string x) → string?

  x : quaternion?/number?
Returns x as a string representation of a quaternion. Given in the form of "(quaternion h i j k)".
>(quaternion->string (quaternion 2 5 34 -1))
"(quaternion 2 5 34 -1)"

procedure

(make-quaternion x) → quaternion?

  x : quaternion?/number?
Only available within the module. Turns any number into a quaternion. If x is a quaternion, returns x. If x is a number, returns (quaternion (real-part x) (imag-part x) 0 0). If neither, returns quaternion 0 0 0 0.
>(make-quaternion 2+3i)
(quaternion 2 3 0 0)

procedure

(unmake-quaternion x) → number?/quaternion?

  x : quaternion?
Only available within the module. If (quaternion-j x) and (quaternion-k x) are zero, returns a complex number, otherwise returns x.
>(unmake-quaternion (quaternion 2 3 0 0))
2+3i

procedure

(scalar-part x) → number?

  x : quaternion?/number?
Returns the real part of the input as a real number.
>(scalar-part (quaternion 2 5 34 -1))
2

procedure

(vector-part x) → quaternion?

  x : quaternion?/number?
Returns the imaginary part of the input as a quaternion with real part 0.
>(vector-part (quaternion 2 5 34 -1))
(quaternion 0 5 34 -1)

procedure

(conjugate w) → quaternion?/number?

  w : quaternion?/number?
Returns the conjugate of w. This is equivilent to (- (real-part w) (vector-part w))]. If none of the arguments are quaternions, uses rackets default conjugate procedure.
>(conjugate (quaternion 2 5 34 -1))
(quaternion  2 -5 -34 1)
}

procedure

(norm w) → number?

  w : quaternion?
(define (quaternion-norm number)
(let ((q (make-quaternion number)))
(sqrt (+ (expt (quaternion-h q) 2)
(expt (quaternion-i q) 2)
(expt (quaternion-j q) 2)
(expt (quaternion-k q) 2)))))
This take a quaternion as input. It squares each component of the quaternion, sums those values and returns the square root of that sum.

procedure

(unit w) → number?

  w : quaternion?
Returns the unit quaternion of w. This is the quaternion divided by its norm.

2.2 Basic Operators

procedure

(+ x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the sum of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default + procedure.
(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(- w) → quaternion?/number?

  w : quaternion?/number?
Returns the opposite of w. The same as (- 0 w)].

procedure

(- x w ...) → quaternion?/number?

  x : quaternion?/number?
  w : quaternion?/number?
Returns the difference of w from x. For multiple ws takes the difference of each w from x working left to right. If none of the xs are quaternions, uses racket’s default - procedure.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

procedure

(* x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the product of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default * procedure.
>(* (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion -122 543 -106 53)

procedure

(/ w) → quaternion?/number?

  w : quaternion?/number?
Returns the reciprocal of w. The same as (/ 1 w)].

procedure

(/ x w ...) → quaternion?/number?

  x : quaternion?/number?
  w : quaternion?/number?
Returns the quotient of x over w. For multiple ws takes the quotient of each w from x working left to right. If any w is exact 0, the exn:fail:contract:divide-by-zero] exn:fail:contract:divide-by-zero exception is raised. If none of the arguments are quaternions, uses racket’s default / procedure.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

2.3 Advanced Operators

procedure

(exp x) → quaternion?/number?

  x : quaternion?/number?
Returns the Euler’s number to the power of x. If x is not a quaternion, uses racket’s default exp procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(expt x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Returns the x to the power of y. If if neither argument is a quaternion, uses racket’s default expt procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(sin x) → quaternion?/number?

  x : quaternion?/number?
Returns the sine of x. If x is not a quaternion, uses racket’s default sin procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(cos x) → quaternion?/number?

  x : quaternion?/number?
Returns the cosine of x. If x is not a quaternion, uses racket’s default cos procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)
