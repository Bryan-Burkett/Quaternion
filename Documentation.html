1 Quaternion

struct

(struct quaternion (h i j k)
  #:extra-constructor-name make-quaternion)
  h : real?
  i : real?
  j : real?
  k : real?
A structure to represent a quaternion. h represents the real part, i, j and 5 represent the imaginary parts.

procedure

(eq? x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Returns true if the two arguments are equevilent mathematically, otherwise returns false. If none of the arguments are quaternions, uses racket’s default eq? function.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)
}

2 Basic Operators

2.1 Addition

procedure

(+ x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the sum of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default + procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

2.2 Subtraction

procedure

(- w) → quaternion?/number?

  w : quaternion?/number?
Returns the opposite of w. The same as (- 0 w).

procedure

(- x w ...) → quaternion?/number?

  x : quaternion?/number?
  w : quaternion?/number?
Returns the difference of w from x. For multiple ws takes the difference of each w from x working left to right. If none of the xs are quaternions, uses racket’s default - function.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

2.3 Multiplication

procedure

(* x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the product of all xs, or for only one x returns x. If none of the arguments are quaternions, uses racket’s default * procedure.
>(* (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion -122 543 -106 53)

2.4 Division

procedure

(/ w) → quaternion?/number?

  w : quaternion?/number?
Returns the reciprocal of w. The same as (/ 1 w).

procedure

(/ x w ...) → quaternion?/number?

  x : quaternion?/number?
  w : quaternion?/number?
Returns the quotient of x over w. For multiple ws takes the quotient of each w from x working left to right. If any w is exact 0, the exn:fail:contract:divide-by-zero exception is raised. If none of the arguments are quaternions, uses racket’s default / function.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

3 Advanced Operators

3.1 Exponentiation

procedure

(exp x) → quaternion?/number?

  x : quaternion?/number?
Returns the e to the power of x. If x is not a quaternion, uses racket’s default exp function.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

procedure

(exp x y) → quaternion?/number?

  x : quaternion?/number?
  y : quaternion?/number?
Returns the x to the power of y. If if neither argument is a quaternion, uses racket’s default expt function.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

3.4 make-quaternion

procedure

(make-quaternion x) → quaternion?

  x : quaternion/number?
Takes any number and returns it as a Quaternion
>(make-quaternion 3)
(quaternion 3 0 0 0)
>(make-quaternion 3+2i)
(quaternion 3 2 0 0)
