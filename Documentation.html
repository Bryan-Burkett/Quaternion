1 Quaternion

struct

(struct quaternion (h i j k)
  #:extra-constructor-name make-quaternion)
  h : number?
  i : number?
  j : number?
  k : number?
A structure to represent a quaternion. h represents the real part, i, j and k represent the imaginary parts.

2 Basic Operators

2.1 Addition

procedure

(+ x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the sum of two or more numbers or quaternions by summing their components, or for only one argument returns that argument. If none of the arguments are quaternions, uses racket’s + procedure.
>(+ (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 1 5 38 15)

2.2 Subtraction

procedure

(- x ...) → quaternion?/number?

  x : quaternion?/number?
Takes the first argument, then takes the difference of all the following arguments from the first arguemnt, or for only one argument returns that argument. If none of the arguments are quaternions, uses racket’s - procedure.
>(- (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion 3 5 30 -17)

2.3 Multiplication

procedure

(* x ...) → quaternion?/number?

  x : quaternion?/number?
Returns the product two or more quaternions or numbers, or for only one argument returns that argument. If none of the arguments are quaternions, uses racket’s * procedure.
>(* (quaternion 2 5 34 -1) (quaternion -1 0 4 16))
(quaternion -122 543 -106 53)

2.4 Subtraction

procedure

(- x ...) → quaternion?/number?

  x : quaternion?/number?
Takes the first argument, then multiplies it by the multaplicative inverse of the remaining arguments, or for only one argument returns that argument. If none of the arguments are quaternions, uses racket’s - procedure.
> (\ (quaternion -122 543 -106 53) (quaternion -1 0 4 16))
(quaternion 2 5 34 -1)

procedure

(make-Quaternion x) → quaternion?

  x : quaternion/number?
Takes any number and returns it as a Quaternion
>(make-quaternion 3)
(quaternion 3 0 0 0)
>(make-quaternion 3+2i)
(quaternion 3 2 0 0)
